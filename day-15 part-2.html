CSS Filter Effects

Explore the power of CSS filters to create stunning visual effects on images without using any image editing software. Hover over each image to see the filter in action.

Blur

Applies a Gaussian blur to the image. The value defines the radius of the blur.

Grayscale

Converts the image to grayscale. 0% is original, 100% is completely grayscale.

Brightness

Adjusts the brightness of the image. 0% is completely black, 100% is normal.

Contrast

Adjusts the contrast of the image. 0% is completely gray, 100% is normal.

Hue Rotate

Applies a hue rotation. The angle defines the number of degrees around the color circle.

Saturate

Adjusts the saturation of the image. 0% is completely desaturated, 100% is normal.

Sepia

Converts the image to sepia. 0% is normal, 100% is completely sepia.

Invert

Inverts the colors in the image. 0% is normal, 100% is completely inverted.

Drop Shadow

Applies a drop shadow effect to the image with specified offset, blur, and color.

Multiple Filters

Combine multiple filters to create unique effects. This example uses contrast, brightness, saturation, and hue-rotate.

How CSS Filters Work

CSS filters allow you to apply graphical effects like blurring, color shifting, and contrast adjustment directly in the browser. These effects are applied to elements before they are displayed.

The filter property accepts one or more filter functions, which are applied in the order provided. You can combine multiple filters to create complex effects.

/* Basic filter syntax */
.element {
  filter: filter-function(value);
}

/* Multiple filters */
.element {
  filter: blur(2px) brightness(150%) contrast(120%);
}

Filters are supported in all modern browsers, though some older browsers may require vendor prefixes. They work on images, backgrounds, and any HTML element.

CSS filters are performance-friendly as they are hardware-accelerated in most browsers, making them a great alternative to image editing for simple effects.
