CSS Day 7: Overflow Property

Overflow Visible: Content goes outside the box if it is too large. This text is slightly longer so it will extend.

Overflow Hidden: Extra content will be clipped and will not show. This text is longer than the container, so you will not see the extra part.

Overflow Scroll: Scrollbars will appear always. This text is long and requires scrolling to view completely. Extra content continues beyond the box height for demonstration.

Overflow Auto: Scrollbar appears only if content overflows. This text will produce scrollbar only if needed.
